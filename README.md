# delta.zsh-theme (oh my zsh theme)

### Installation

```
$ git clone https://github.com/dongri/delta-zsh-theme.git
$ ln -s $PWD/delta-zsh-theme/delta.zsh-theme ~/.oh-my-zsh/custom/themes/delta.zsh-theme
```
